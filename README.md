# 신정개발 경영지원 대시보드

(주)신정개발(무인로봇 · 하수도 준설 · 화학설비 크리닝)의 경영지원 업무용 대시보드 웹서비스입니다.
경영지원 담당자가 관리하는 엑셀 파일(`sj-management.xlsx`)을 브라우저에서 직접 읽어
매출 · 매입 · 프로젝트 · 자금 현황을 시각화합니다.

## 🔒 보안 구조 (가장 중요)

**데이터 파일은 절대 이 저장소에 올라가지 않습니다.**

- 엑셀 원본은 사용자 PC에만 보관합니다. `.gitignore`가 `*.xlsx` 등 모든 스프레드시트
  파일의 커밋을 차단합니다.
- 웹앱은 **서버 없이** 동작합니다. 사용자가 브라우저에서 파일을 선택하면
  [SheetJS](https://sheetjs.com)가 브라우저 메모리 안에서만 파싱하며,
  네트워크로 전송되는 데이터는 없습니다.
- 파싱 결과와 [엑셀 다운로드]용 원본 사본은 해당 브라우저의 `localStorage`에만 저장되며,
  화면의 [데이터 지우기] 버튼으로 즉시 삭제할 수 있습니다.
- 급여 · 연락처 · 이메일 등 민감정보는 화면에 표시하지 않습니다
  (`src/lib/parse.ts`의 인사 시트 매핑 참고).

## ✏️ 편집과 엑셀 다운로드

- 매출 · 매입 · 프로젝트 · 자금 탭에서 행 **추가 / 수정 / 삭제**가 가능합니다.
  부가세(10%) · 합계 · 자금 잔액 · 프로젝트 기성/수금은 자동 재계산됩니다.
- 우측 상단 **[엑셀 다운로드]**를 누르면 수정 내용이 반영된
  `sj-management_날짜.xlsx` 파일이 이 컴퓨터에 저장됩니다.
- 다운로드 파일은 편집한 4개 시트(매출관리·매입관리·프로젝트·자금관리)만 새로 쓰고,
  **인사 · 거래처 · 기준정보 · 월별요약 등 나머지 시트는 원본 그대로 보존**하므로
  원본 파일을 대체하는 새 마스터로 사용할 수 있습니다.
  (단, SheetJS 무료판 한계로 셀 색상·드롭다운 등 일부 서식은 포함되지 않습니다.)

## 실행

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # dist/ 에 정적 파일 생성
```

## Vercel 배포

1. Vercel에서 **Add New → Project** 후 이 저장소(`sj-jhs`)를 Import
2. Framework Preset: **Vite** (자동 감지) — 추가 설정 없이 Deploy
3. 배포된 주소에 접속해 각자 PC의 엑셀 파일을 불러와 사용

정적 파일만 배포되므로 Vercel에도 회사 데이터는 존재하지 않습니다.

## 엑셀 파일 규칙

파일 어디에 있든 상관없으며(권장: 회사 공유폴더 또는 담당자 PC), 아래 시트 구조를 따라야 합니다.

| 시트명 | 필수 | 내용 |
|---|---|---|
| 매출관리 | ✅ | 일자 · 사업부문 · 거래처명 · 프로젝트코드 · 공급가액 · 수금상태 · 수금액 등 |
| 매입관리 | ✅ | 일자 · 비용구분 · 거래처명 · 공급가액 · 지급상태 등 |
| 프로젝트 | | 프로젝트코드 · 계약금액 · 진행상태 · 진행률 등 (기성/수금은 매출에서 자동 집계) |
| 자금관리 | | 일자 · 구분(입금/출금) · 입금액 · 출금액 (잔액은 자동 재계산) |
| 인사 | | 성명 · 부서 · 재직상태 등 인원 현황용 최소 정보만 사용 |
| 거래처 | | 거래처코드 · 거래처명 · 구분 |

- 헤더는 1행에 위치해야 하며, 열 이름은 접두어로 인식합니다. (예: `공급가액(원)` → `공급가액`)
- 날짜는 엑셀 날짜 형식 또는 `yyyy-mm-dd` 문자열 모두 지원합니다.
- 부가세/합계/잔액 수식 열은 값이 비어 있어도 앱이 자동 계산합니다.

## 기술 스택

- Vite + React 18 + TypeScript
- SheetJS (xlsx) — 클라이언트 사이드 엑셀 파싱
- Recharts — 차트

## 폴더 구조

```
src/
  lib/parse.ts      # 엑셀 → 데이터 모델 파싱 (민감정보 제외)
  lib/aggregate.ts  # KPI/월별/부문별 집계
  lib/storage.ts    # localStorage 저장/삭제
  components/       # 파일 업로드, KPI 카드, 상태 배지
  views/            # 대시보드 · 매출 · 매입 · 프로젝트 · 자금 탭
```
